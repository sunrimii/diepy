# 遊戲說明
1. 模式分為單人和多人
2. 棕色母艦為主要敵人, 皆擊墜即獲勝
3. 黃色補包可獲能力點數, 皆以5點為上限
    - 發射彈藥數
    - 彈藥傷害
    - 彈藥速度
    - 冷卻時間
    - 移動速度
4. 死亡會遺失所有能力點數

# 執行步驟
`$ pip -r requirements.txt`
`$ python game.pyw`
